# Primitive Data Types in Java and their sizes

- byte
  - 8 bits and signed
- short
  - 16 bits and signed
- char
  - 16 bits and unsigned, so that it may represent Unicode characters
- int
  - 32 bits and signed
- long
  - 64 bits and signed
- float
  - 32 bits and signed
- double
  - 64 bits and signed
- boolean
  - it's not numeric, may only have true or false values
