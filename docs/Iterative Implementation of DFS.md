# Iterative Implementation of DFS

```python
marked = [False] * G.size()
def dfs(G,v):
    stack = [v]
    while len(stack) > 0:
        v = stack.pop()
        if not marked[v]:
            visit(v)
            marked[v] = True
            for w in G.neighbours(v):
                if not marked[w]:
                    stack.append(w)
```

Time complexity: $O(V+E)$;
$V$ = no. of vertices
$E$ = no. of edges

More generalizable than [[Recursive Implementation of DFS]].
